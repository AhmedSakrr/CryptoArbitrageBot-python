An arbitrage bot to find price descrepencies between crytocurrency exchanges automatically. 

![Preview](https://user-images.githubusercontent.com/66185698/147394831-6377de38-55b3-4ec9-88d2-399f65a8ddc4.png)

Supported exchanges:

- Coinbase
- Bitfinex
- Bitstamp
- Kraken
- CEX
- BL30
- Bitbay
- Liquid
